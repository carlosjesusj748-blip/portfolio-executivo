Sobre este Portfólio

Este portfólio não é apenas uma página estática, mas uma demonstração prática das minhas habilidades de orquestração técnica. Ele foi arquitetado e codificado do zero em parceria com Agentes Autônomos de IA (Google DeepMind / Gemini). A estrutura de automação de contato envia os leads instantaneamente para um fluxo do Make, que é processado via API pela inteligência do Groq (Llama 3.3), consolidando o conceito de Agentic Workflow e escalabilidade cloud-native.
